# AICLAWCHUHAI 官网

可直接部署到 GitHub Pages 或 Cloudflare Pages 的静态站点。

## 文件结构
- `index.html` 首页
- `styles.css` 样式
- `articles/` 文章页
- `robots.txt`
- `sitemap.xml`

## GitHub + Cloudflare Pages 部署
1. 新建 GitHub 仓库，把全部文件上传到仓库根目录
2. 打开 Cloudflare Pages
3. 连接 GitHub 仓库
4. Framework preset 选择 `None`
5. Build command 留空
6. Output directory 填 `/`
7. 绑定域名 `aiclawchuhai.shop`
